# Demo 微服务项目

这是一个基于 Spring Cloud 2021.0.4 和 Spring Boot 2.7.9 的现代化微服务/多模块项目，采用前后端分离架构，集成了多种主流中间件和开发工具。

## 📖 项目简介

本项目旨在提供一个高性能、可扩展的微服务基础架构，涵盖了网关治理、系统管理、业务管控等核心功能。

## 📂 项目结构

```text
demo/
├── common/                 # 公共模块：通用工具类、统一异常处理、核心配置
├── gateway/                # 网关模块：请求路由、负载均衡、限流熔断、安全过滤
├── system/                 # 系统模块：用户权限、菜单管理、字典配置、系统监控
├── manage/                 # 管理模块：业务管理功能实现
├── frontend/               # 前端项目 (如果存在)
├── pom.xml                 # 父项目 POM：版本锁定与公共依赖管理
└── README.md               # 项目说明文档
```

## 🚀 核心模块说明

### 1. Common 模块 (公共核心)
- **统一响应封装**：`Result<T>` 统一接口返回格式，提升前后端交互效率。
- **全局异常处理**：自研异常拦截机制，确保系统鲁棒性。
- **核心工具类**：继承 Hutool、FastJSON 等优秀开源库。
- **安全加固**：包含 XSS 过滤、SQL 注入拦截等核心安全组件。

### 2. Gateway 模块 (服务网关)
- **动态路由**：基于 Nacos 实现服务的注册与发现，支持动态扩缩容。
- **认证授权**：内置 `AuthFilter` 全局认证过滤器，支持白名单与 JWT 验证。
- **流量治理**：集成 Sentinel 路由限流，基于 Redis 的多维度（IP/用户）流控。
- **API 文档聚合**：集成 Knife4j，实现微服务 API 接口的动态展示与调试。

### 3. System 模块 (权限管理)
- **RBAC 模型**：完整的用户、角色、菜单权限管理体系。
- **审计日志**：自动记录操作日志与登录行为。
- **配置管理**：动态参数配置，无需重启服务即可生效。

### 4. Manage 模块 (业务逻辑)
- 承载核心业务逻辑，遵循阿里巴巴 Java 开发手册，代码结构清晰。

## 🏁 快速开始

### 环境依赖
- JDK 1.8+ (强制指定)
- Maven 3.6+
- Redis
- MySQL 8.0+
- Nacos (用于服务发现与配置)

### 编译与运行
1. **基础环境**：确保上述中间件已在本地或 Docker 中启动并运行。
2. **构建项目**：
   ```bash
   mvn clean install -DskipTests
   ```
3. **启动顺序**：
   - 启动 `Nacos` 控制台
   - 启动 `SystemApplication` (端口: 8080 - 示例)
   - 启动 `GatewayApplication` (端口: 9999)

### 访问入口
- **网关地址**：`http://localhost:9999`
- **在线文档**：`http://localhost:9999/doc.html` (Knife4j)

## 规范与建议

- **编码规范**：遵循 《Alibaba Java Coding Guidelines》。
- **接口开发**：请求统一使用 `GET/POST`，通过 DTO 接收参数，VO 返回数据。
- **测试建议**：代码提交前请确保单元测试通过。
- **注释要求**：所有类、方法需包含详尽的 Javadoc 中文注释。
